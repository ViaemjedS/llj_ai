 - stylus 何物？
    - 是一个css预处理器
    - 更快，更专业
    - .styl 后缀的文件
    - 编译成css

- stylus 是css 的超集
  浏览器还是只认css
  stylus common.styl -o common.css 手动转换
  stylus -w common.styl -o common.css 自动转换成css

  body color 子元素 body
  有些也不会继承

- background-size: cover 等比例缩放， 裁剪
